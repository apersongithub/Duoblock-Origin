Compatibility tips

This extension is built to survive many changes of Duolingo's web interface, but sometimes site changes may require adjustments. If you notice any issues, try switching your block mode.

Note: Soft block may have issues if you use Duolingo Max or Super. In that case, try Hard or Extreme block modes.

Modes

Blocking Mode
